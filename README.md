## 🚦 Project Rules

 # 개발환경
 - OS: Window / Mac
 - Code editor: Visual Studio Code
 - Client-Tool : Insomnia
 - DB-Tool: DBeaver
 - Database: AWS/RDS (MySQL)
 - Server: AWS/EC2

 # 개발언어
 - Front-End : Html, CSS, Javascript
 - Back-End : Javascript
    - Node.js, Express.js
 - Database: MySQL
    - ORM: Prisma
